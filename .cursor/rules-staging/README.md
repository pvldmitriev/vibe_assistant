# Rules Staging Directory

Эта папка используется для создания и редактирования правил Cursor AI.

## Назначение

- Рабочая область для создания новых правил
- Временное хранилище при редактировании существующих правил
- Изоляция от финальных правил в `.cursor/rules/`

## Workflow

См. подробное описание в файле `/Users/paveldmitriev/vibe_assistant/.cursor/rules/rules-editing-workflow.mdc`

## Важно

- Файлы в этой папке — временные
- После перемещения в `.cursor/rules/` файлы можно удалять
- Не коммитить промежуточные файлы в git
