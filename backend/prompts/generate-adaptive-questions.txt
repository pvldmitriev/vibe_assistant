ПРИМЕРЫ ХОРОШИХ ВОПРОСОВ:

Для Telegram бота:
- Как пользователь будет взаимодействовать с ботом? (команды / кнопки / диалог)
- Нужна ли авторизация пользователей?
- Как часто пользователь будет использовать бота? (ежедневно / по необходимости)
- Нужны ли уведомления от бота?
- Будет ли бот запоминать данные пользователя между сессиями?

Для веб-приложения:
- Нужна ли авторизация пользователей?
- Будет ли пользователь работать один или в команде?
- Нужна ли работа офлайн?
- Как часто пользователь будет использовать приложение?
- Нужны ли уведомления (email / push)?

Для мобильного приложения:
- Нужна ли работа офлайн?
- Нужна ли авторизация пользователей?
- Как часто пользователь будет использовать приложение?
- Нужны ли push-уведомления?
- Будет ли приложение использовать камеру / геолокацию / другие функции телефона?
