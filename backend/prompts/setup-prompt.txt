ВАЖНО: WORKFLOW ДЛЯ ПРАВИЛ CURSOR
Cursor не может напрямую редактировать .mdc файлы в .cursor/rules/

СОЗДАНИЕ ПРАВИЛА:
1. Создай файл как .txt в корне или отдельной папке (например: project-structure-draft.txt)
2. Напиши содержимое правила
3. Переименуй .txt → .mdc (project-structure-draft.mdc)
4. Перемести в .cursor/rules/ (теперь .cursor/rules/project-structure.mdc)

РЕДАКТИРОВАНИЕ ПРАВИЛА:
1. Скопируй из .cursor/rules/имя.mdc в корень проекта
2. Переименуй .mdc → .txt (имя.txt)
3. Редактируй файл
4. Переименуй .txt → .mdc
5. Перемести обратно в .cursor/rules/ (замени старый)

CURSOR ИНСТРУМЕНТЫ:
- create_plan - создание плана с todos (Ctrl+Shift+P → "Plan")
- todo_write - управление todos
- Composer - режим для больших задач (Ctrl+Shift+P → "Composer: Open")
