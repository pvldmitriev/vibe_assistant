## Исправленные проблемы

### ✅ ИСПРАВЛЕНО: Критический баг - форма не отображается для категорий "Telegram бот" и "Мобильное приложение"

**Проблема:** На шаге 3 форма "Базовые вопросы" не рендерилась для категорий BOT и MOBILE_APP.

**Исправление:** Базовые вопросы теперь загружаются всегда, независимо от восстановления сессии.

### ✅ ИСПРАВЛЕНО: Проблема с валидацией на шаге 2

**Проблема:** Кнопка "Далее" оставалась disabled даже после ввода валидной идеи.

**Исправление:** Добавлена проверка минимальной длины идеи (20 символов).
